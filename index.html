ABOMINATOR
a tool for making an image or video ABOMINABLE, with

example texture
audio, auto-detected. This tool forces user interaction if the resource is a video, required by modern browsers for playing audio.



  URL cheatsheet
  e.g. ... /?url=res.jpg&flyers=3&random&speed=2 ...

   flyers - number of ABOMINATORS [0 - 15]
  flyersize - size of ABOMINATORS [0 - inf]
  speed - animation master throttle [0 - inf]
  genScale - default level of distortion [0 - 1]
  random - random ABOMINATOR motion paths
  gamma - adjust brightness [0 - inf]
  url - link to custom resource [image or video]
  playbackspeed - if video, speed [1 = normal, 0 - 10]
  loop - if video, loop [true (default) or false]
  autoplay - if video, play muted, no button prompt
